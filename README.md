# Angimotti Ledger SideStore

Public SideStore distribution files for Angimotti Ledger iOS.

## Source

Use this URL in SideStore:

```text
https://kkzzxxcc.github.io/angimotti-ledger-sidestore/source.json
```

## Contents

- `source.json`: SideStore source metadata.
- `icon.png`: App icon used by the source.
- GitHub Releases: IPA assets for each released version.

This repository intentionally does not contain app source code, Xcode project files, signing files, provisioning profiles, credentials, or private development data.
